Seja bem vindo (a) ao DS Game List

Este é um projeto de estudos, tendo como finalidade a construção de um sistema de login de usuários

Tais usuários podem adicionar jogos ao sistema; Criar listar personalizadas de jogos e incluí-los na mesma; deletar jogos e listas; e também atualizar ambos.

O sistema de login conta com: Cadastro de usuários com envio de email para ativação de conta; Envio de email para modificação de senha; Login facilitado (OAuth2)

Também há dois tipos de papéis de usuário: Administrador, podendo deletar usuários do sistema; e usuário que não tem tal privilégio
